"Funshoppers" is an e-commerce smart application which comes with beacon technology.

When the application starts it needs to know if you are in a mall(where beacons will work).If not in mall, customer can buy products based on cash on delivery.

This application can scan, locate and track bluetooth Low Energy beacons (Eddystone or AltBeacons), which allows you to define actions that will be triggered when 
a specified event occurs(eg:When you are within a range of beacon).
These events are supported:
• Beacon enters a region (beacon in range)
• Beacon leaves the region (beacon is out of range)
• Beacon is near you.
